# Creating an Account

Anarchy AI works best with an account for cloud features and API access.

## Why Create an Account?

### Benefits
- ✅ Cloud sync across devices
- ✅ Save projects online
- ✅ Access to premium models
- ✅ Priority generation queue
- ✅ Share workflows with team
- ✅ Backup and restore

### Free vs Pro

| Feature | Free | Pro |
|---------|------|-----|
| Generations/day | 50 | Unlimited |
| Cloud storage | 1 GB | 100 GB |
| Max resolution | 1024x1024 | 2048x2048 |
| Queue priority | Standard | High |
| Team sharing | 3 members | Unlimited |
| API access | Limited | Full |
| Custom models | ❌ | ✅ |

## Sign Up

### Step 1: Open Sign-up
- Launch Anarchy AI
- Click **"Create Account"** on welcome screen
- Or go to **Help → Sign Up**

### Step 2: Enter Details
- **Email**: Valid email address
- **Password**: 8+ characters, mix of letters/numbers
- **Username**: Public display name
- **Organization** (optional): Company or team name

### Step 3: Verify Email
- Check your inbox for verification email
- Click the verification link
- Return to Anarchy AI

### Step 4: Configure API
- Enter your Replicate API key
- Or use Anarchy AI credits
- Test connection

## Sign In

### Already Have Account?
1. Click **"Sign In"** on welcome screen
2. Enter email and password
3. Click **Sign In**

### Stay Signed In
- Check "Remember me" to stay logged in
- Your projects sync automatically

### Forgot Password?
1. Click **"Forgot Password"**
2. Enter your email
3. Follow reset instructions in email

## Account Settings

Access via **Settings → Account**:

### Profile
- Change username
- Update avatar
- Edit bio
- Set timezone

### Security
- Change password
- Enable 2FA (Two-Factor Authentication)
- View login history
- Manage sessions

### Billing
- View usage statistics
- Upgrade to Pro
- Manage payment methods
- View invoices

### API Keys
- Generate new API keys
- Revoke old keys
- Set rate limits
- View usage logs

## Cloud Sync

### Enable Sync
Settings → Cloud → Enable Sync

### What Syncs
- All projects (.ana files)
- History and generations
- Settings and preferences
- Custom prompts/templates

### Sync Status
- Green dot: Synced
- Yellow dot: Syncing
- Red dot: Error (check connection)

## Team Collaboration

### Create Team
1. Account → Teams → Create Team
2. Set team name
3. Invite members by email
4. Assign roles (Admin/Editor/Viewer)

### Sharing Projects
1. Open project
2. File → Share → Share with Team
3. Set permissions
4. Send link or notify team

### Version History
- Track changes by team members
- Revert to previous versions
- Comment on changes
- Lock projects while editing

## Privacy & Data

### Your Data
- Projects stored encrypted
- Images processed securely
- No training on your data
- Delete anytime

### Export Data
Account → Privacy → Export Data
- Download all your projects
- Export history as archive
- Portable format

### Delete Account
Account → Privacy → Delete Account
- Permanent deletion
- 30-day grace period
- Data removed from servers

## Troubleshooting

### Can't Sign In
- Check email/password
- Verify email if new account
- Reset password if forgotten
- Check internet connection

### Sync Not Working
- Check sync status indicator
- Sign out and sign in again
- Check storage limit
- Verify internet connection

### API Key Issues
- Verify key is valid at replicate.com
- Check key hasn't expired
- Ensure sufficient credits
- Try regenerating key
