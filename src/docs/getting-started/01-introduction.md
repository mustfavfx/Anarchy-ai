# Introduction to Anarchy AI

**Anarchy AI** is a powerful node-based AI image generation and workflow automation tool designed specifically for architects, designers, and creative professionals.

## What You Can Do

- 🎨 **Generate AI Images** from text prompts or existing images
- 🔗 **Create Workflows** by connecting nodes together
- 🏗️ **Architectural Visualization** with specialized models
- 📤 **Export Results** to professional CAD/BIM software
- 📚 **Organize History** and manage your generated assets

## Core Concepts

### Nodes
Nodes are the building blocks of your workflow:
- **Source Nodes**: Start with an image or prompt
- **Ghost Nodes**: AI processing steps
- **Result Nodes**: Final outputs

### Canvas
The infinite canvas where you build your workflows:
- Drag to pan
- Scroll to zoom
- Right-click for context menu

### History
Every generation is automatically saved:
- Browse by source image
- Search and filter results
- Send back to canvas anytime

## Quick Start

1. **Add Source** - Drag an image or click "+ Source"
2. **Add Processing** - Right-click a node and select processing type
3. **Generate** - Click "Generate" and wait for AI
4. **Export** - Right-click results to save or export

## Tips for Best Results

- Use clear, descriptive prompts
- Start with good quality source images
- Experiment with different AI models
- Save your workflows as `.ana` files
