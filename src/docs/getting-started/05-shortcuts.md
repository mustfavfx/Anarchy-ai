# Keyboard Shortcuts

Master these shortcuts to work faster in Anarchy AI.

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| Ctrl+N | New Project | Clear canvas and start fresh |
| Ctrl+S | Save | Save project to current file |
| Ctrl+Shift+S | Save As | Save with new name/location |
| Ctrl+O | Open | Load existing .ana project |
| Ctrl+Shift+O | Import | Import image or workflow |

## Edit Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| Ctrl+Z | Undo | Reverse last action |
| Ctrl+Y | Redo | Repeat undone action |
| Ctrl+X | Cut | Cut selected node(s) |
| Ctrl+C | Copy | Copy selected node |
| Ctrl+V | Paste | Paste node or image |
| Delete | Delete | Remove selected node |

## Canvas Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| Space + Drag | Pan | Move around canvas |
| Ctrl + Scroll | Zoom | Zoom in/out |
| 0 (zero) | Reset Zoom | Fit to 100% |
| F | Fit All | Show all nodes |
| Home | Center | Focus on selected node |

## Node Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| Tab | New Ghost | Add child to selected |
| Enter | Edit | Edit selected node |
| Esc | Deselect | Clear selection |
| Ctrl + A | Select All | Select all nodes |
| Ctrl + D | Duplicate | Clone selected node |

## Generation

| Shortcut | Action | Description |
|----------|--------|-------------|
| Ctrl + G | Generate | Start AI generation |
| Ctrl + . | Cancel | Stop current generation |
| Ctrl + R | Retry | Regenerate selected node |
| Ctrl + Enter | Quick Gen | Generate with defaults |

## View & Interface

| Shortcut | Action | Description |
|----------|--------|-------------|
| F11 | Full Screen | Toggle fullscreen |
| Ctrl + B | Sidebar | Show/hide sidebar |
| Ctrl + H | History | Open history panel |
| Ctrl + Shift + H | History Modal | Full history view |
| F1 | Help | Open documentation |
| Ctrl + , | Settings | Open preferences |

## Selection

| Shortcut | Action | Description |
|----------|--------|-------------|
| Click | Select | Select single node |
| Shift + Click | Multi-Select | Add to selection |
| Ctrl + Click | Toggle | Add/remove from selection |
| Drag Box | Area Select | Select multiple nodes |
| Esc | Clear | Deselect all |

## Export

| Shortcut | Action | Description |
|----------|--------|-------------|
| Ctrl + E | Export | Export selected image |
| Ctrl + Shift + E | Export All | Batch export |
| Ctrl + P | Print | Print or export PDF |

## Special

| Shortcut | Action | Description |
|----------|--------|-------------|
| Ctrl + Shift + D | Debug | Open developer tools |
| Ctrl + / | Comment | Toggle prompt comment |
| Ctrl + F | Find | Search in canvas |
| Ctrl + Shift + F | Global Find | Search all projects |

## macOS Equivalents

| Windows | macOS | Action |
|---------|-------|--------|
| Ctrl | Cmd (⌘) | Primary modifier |
| Alt | Option (⌥) | Secondary modifier |
| Ctrl+Z | Cmd+Z | Undo |
| Ctrl+S | Cmd+S | Save |

## Custom Shortcuts

You can customize shortcuts in **Settings → Keyboard**:

1. Click action you want to change
2. Press new key combination
3. Click "Save"
4. Conflicts will be highlighted

## Learning Shortcuts

Tips for memorization:
1. Start with File operations (N, S, O)
2. Learn Canvas navigation (Space, Scroll, F)
3. Add Edit commands (Z, Y, C, V)
4. Practice daily for muscle memory
