# Installation & Setup

## System Requirements

### Minimum Requirements
- **OS**: Windows 10/11, macOS 12+, or Linux
- **RAM**: 8 GB
- **Storage**: 2 GB free space
- **Internet**: Required for AI generation

### Recommended
- **RAM**: 16 GB or more
- **GPU**: For faster local processing
- **Display**: 1920x1080 or higher

## Installation

### Windows
1. Download the installer from [anarchy-ai.com](https://anarchy-ai.com)
2. Run `AnarchyAI-Setup.exe`
3. Follow the installation wizard
4. Launch from Start Menu or Desktop

### macOS
1. Download `AnarchyAI.dmg`
2. Drag to Applications folder
3. Allow in System Preferences → Security
4. Launch from Applications

### Linux
```bash
# Download AppImage
chmod +x AnarchyAI.AppImage
./AnarchyAI.AppImage
```

## Initial Setup

### 1. API Key Configuration
On first launch, you'll need to configure your AI provider:

**Replicate (Recommended)**
1. Visit [replicate.com](https://replicate.com)
2. Create account and get API token
3. Paste in Anarchy AI Settings

**Alternative Providers**
- Stability AI
- OpenAI (DALL-E)
- Midjourney (via API)

### 2. Workspace Setup
Choose your default workspace folder:
- Projects will be saved here
- History database stored here
- Export outputs go here

### 3. Interface Customization
- Select theme (Dark/Light)
- Choose default AI model
- Set keyboard shortcuts

## Verifying Installation

Create a test project:
1. Click **New Project** (Ctrl+N)
2. Add a source image or prompt
3. Click **Generate**
4. Check if image generates successfully

## Troubleshooting

### App Won't Start
- Check system requirements
- Update graphics drivers
- Run as administrator (Windows)

### Generation Fails
- Verify API key is correct
- Check internet connection
- Try different AI model

### Performance Issues
- Close other applications
- Reduce canvas complexity
- Enable virtualization in settings

## Updates

Anarchy AI checks for updates automatically. You can also:
- **Manual Check**: Help → Check for Updates
- **Auto-Update**: Enable in Settings
