# Anarchy AI

**The Professional AI Operating System for Architects**

Anarchy AI is a desktop-first architectural AI platform that combines visual workflow building, project memory, image lineage tracking, multi-model orchestration, and advanced architectural visualization into a single integrated environment.

Unlike traditional AI image generators that operate as isolated prompt-to-image tools, Anarchy AI enables architects and visualization studios to build reusable AI workflows, track the evolution of every design decision, restore complete generation trees, and manage entire projects from concept exploration to final presentation.

Built specifically for architects, designers, visualization artists, and creative studios.

---

## Why Anarchy AI

### Visual Workflow Builder

Design AI pipelines visually using a node-based canvas. Connect source images, prompts, models, variations, and edits into reusable workflows.

### Interactive History Graph

Explore image generations as a branching visual tree rather than a flat gallery. Understand how every image evolved from its origin.

### Full Workflow Restoration

Load an entire generation workflow back into the Builder with a single click, including nodes, connections, prompts, models, and generation history.

### Project Memory

Keep generations, workflows, references, and creative exploration attached to projects instead of scattered across folders.

### Multi-Model Architecture

Combine multiple AI engines inside the same workflow and route outputs between models for advanced creative pipelines.

### Image Lineage Tracking

Every generation preserves its ancestry, allowing complete traceability from the original source image to the final result.

### Local-First Design

Projects, workflows, history, and assets remain under user control with local storage architecture and optional cloud services.

### Built for Architectural Visualization

Optimized specifically for architectural rendering, concept development, design exploration, visualization enhancement, and presentation workflows.

## Core Modules

### Builder

A professional node-based visual workflow editor for AI-powered image generation and editing.

### History

A complete visual memory system with timeline view, lineage view, workflow graph exploration, search, filtering, and restoration.

### Library

A centralized asset management system for generated images, references, project resources, and reusable content.

### Projects

Organize workflows, generations, references, and creative exploration into structured project environments.

### AI Assistant

Architectural AI assistant designed to support visualization workflows, design decisions, and creative exploration.

### Integrations

Support for architectural and visualization ecosystems, enabling future connectivity with professional design software.
